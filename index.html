Flight Plan

Suggested 15-Class Flow

Week 1
Classes 1-2 · Kickstart Python, environment setup, and master the foundational data types. Ideal for learners new to Python or refreshing fundamentals.
Week 2
Classes 3-4 · Control flow mastery and modular thinking. Build confidence writing reusable functions alongside supportive exercises.
Week 3
Classes 5-6 · Deep dive into data structures and persistent storage. Transition from scripts to data-driven utilities.
Week 4
Classes 7-8 · Error handling best practices and first OOP principles. Prepare for building larger systems.
Week 5
Classes 9-10 · Sharpen OOP design. Debug, refactor, and package your code base for collaboration.
Week 6
Classes 11-12 · Learn to integrate with real APIs and step into Django backend patterns. Connect Python to the web.
Week 7
Class 13 · Engage with AI tooling using Python. Experiment with LLMs and retrieval-augmented generation ideas.
Week 8
Classes 14-15 · Capstone build, presentations, feedback loop, and pathway to career-focused projects.
FAQ & Tips

How to Get the Most From This Module

Do I need to register to view modules?

No. The full curriculum is open. Registration just helps the team track interest and send you cohort updates.

How is progress stored?

We use browser localStorage. Checkboxes are always available and persist on this device only. Use the clear button if you want to reset.

What tools should I prepare?

Install Python 3.x, VS Code or PyCharm, plus Git. By Module 4 you will add Django, requests, and other packages.

Can I preview Django content now?

Absolutely. Skim Module 4, Class 12 and follow the activity prompts. If you're a Flutter dev, think of Django views as controllers and templates as UI widgets.
